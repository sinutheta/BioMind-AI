## рҹҡЁ The Problem

> **Heart disease kills someone every 33 seconds. Type 2 diabetes affects 1 in 10 adults globally. Most people only find out when it's too late.**

Traditional health screening is:
- **Fragmented** вҖ” you see a GP for one thing, a specialist for another
- **Reactive** вҖ” you act after symptoms become serious
- **Opaque** вҖ” you get a diagnosis with no explanation you can actually understand
- **Inaccessible** вҖ” expensive, slow, geography-dependent

**BioMind AI changes all of that. In under 5 minutes.**

---

## рҹ’Ў The Solution

BioMind AI is a **unified, multi-model health risk detection platform** that fuses three independent AI engines into one seamless assessment. You describe your symptoms, enter clinical vitals, and share your lifestyle вҖ” and BioMind returns:

- рҹҺҜ **Risk scores** across 41 diseases, ranked by probability
- вқӨпёҸ **Heart disease probability** from your clinical data
- рҹ©ё **Diabetes risk** from your metabolic profile
- рҹ§¬ **A personalised lifestyle multiplier** вҖ” because how you live matters as much as what you feel
- рҹ—ЈпёҸ **Plain-English AI explanation** вҖ” no medical degree needed

## рҹӨ– The 3 AI Models вҖ” Under the Hood

### Model 1 В· Symptom Classifier
> *Random Forest В· DDXPlus-style dataset В· 132 binary features вҶ’ 41 diseases*

The backbone of BioMind. Trained on a rich multi-label symptom dataset, this Random Forest processes 132 checkbox symptoms вҖ” from `high_fever` and `breathlessness` to `polyuria` and `blurred_and_distorted_vision` вҖ” and maps them to **41 possible diagnoses** including Heart Attack, Tuberculosis, Dengue, Malaria, Diabetes, and more.

### Model 2a В· Heart Risk Engine
> *XGBoost В· Clinical vitals В· StandardScaler preprocessing*

A gradient-boosted model trained on cardiovascular data. Takes clinical inputs (blood pressure, cholesterol, chest pain type, resting ECG, etc.), scales them through `scaler_heart.pkl`, and outputs a heart disease probability score.

### Model 2b В· Diabetes Risk Engine
> *XGBoost В· Metabolic panel В· StandardScaler preprocessing*

A companion XGBoost model trained on metabolic/glycaemic data. Uses `scaler_diab.pkl` and outputs a diabetes probability, which is then cross-validated against symptom model predictions (polyuria, excessive hunger, etc.) for higher confidence.

### Model 3 В· Lifestyle Multiplier
> *Custom weighted scoring В· 7 behavioural factors В· 0.5Г— вҖ“ 2.5Г— modifier*

This is what makes BioMind different. Your lifestyle isn't ignored вҖ” it's **quantified**. Sleep hours, stress level, exercise frequency, smoking status, alcohol consumption, diet quality, and water intake all feed into a modifier that scales every disease score up or down. Live well вҶ’ your risks reduce. Consistently.

## рҹ—ЈпёҸ AI Explainability Layer

After all models run, **Claude AI** (with GPT-4o fallback) receives your full risk profile and generates a personalised, plain-English explanation:

- *Why* your top risks scored the way they did
- What specific symptoms or clinical values are most significant
- What lifestyle changes could have the highest impact
- **Red flags** вҖ” symptoms or combinations that warrant urgent medical attention

No jargon. No confusion. Just clarity.

## рҹҡҖ Quick Start

### Prerequisites
- Python 3.11+
- Node.js 18+
- Git

### 1 В· Clone the repo
```bash
git clone https://github.com/sinutheta/BioMind-AI.git
cd BioMind-AI
```

### 2 В· Start the Backend
```bash
cd backend

# Activate virtual environment (Windows)
.\venv\Scripts\Activate.ps1

# Or on Mac/Linux
source venv/bin/activate

# Run the server
python -m uvicorn app.main:app --reload --port 8000
```
> Backend live at **http://localhost:8000**
> API docs at **http://localhost:8000/docs**

### 3 В· Start the Frontend
```bash
# In a new terminal
cd frontend
npm install
npm run dev
```
> App live at **http://localhost:5173**
